Getting Started with E2P Simulator

Welcome to the E2P Simulator! This guide will help you understand how to use this tool to explore the relationship between statistical effect sizes and their practical predictive utility.

What is the E2P Simulator?

The E2P Simulator (Effect-to-Prediction Simulator) is an interactive tool that helps researchers understand how statistical effect sizes translate into practical predictive utility. It visualizes the relationship between commonly reported statistical metrics (such as Cohen's d and correlation coefficients) and their real-world predictive value, while accounting for crucial factors like measurement reliability and outcome base rates.

Choosing Your Analysis Mode

The E2P Simulator offers two analysis modes:

Binary Mode

Use this mode when your groups are naturally distinct (e.g., treatment vs. control, male vs. female).

This mode helps you understand:

  • How Cohen's d values translate to group separation
  • How reliability affects observed effect sizes
  • How base rates impact classification metrics
  • How many predictors would be needed to achieve desired group separation

Continuous Mode

Use this mode when your groups are formed by thresholding a continuous outcome variable (e.g., responders vs. non-responders).

This mode helps you understand:

  • How correlation coefficients translate to predictive performance
  • How reliability affects observed correlations
  • How base rates impact classification metrics
  • How many predictors would be needed to achieve desired explained variance

Using the Binary Mode

In Binary mode, you can:

  1. Adjust Cohen's d: Use the slider to set different effect sizes and observe how they affect group separation.
  2. Modify reliability: Adjust the reliability sliders to see how measurement error affects observed effect sizes.
  3. Change base rates: Adjust the percentage of cases in each group to see how prevalence affects classification metrics.
  4. View classification metrics: See how your settings affect accuracy, sensitivity, specificity, and other metrics.
  5. Use the Mahalanobis D calculator: Estimate how many predictors of smaller effect sizes would need to be combined to achieve a desired level of group separation.

Using the Continuous Mode

In Continuous mode, you can:

  1. Adjust correlation coefficients: Use the slider to set different correlation strengths and observe how they affect predictive performance.
  2. Modify reliability: Adjust the reliability sliders to see how measurement error affects observed correlations.
  3. Change base rates: Adjust the percentage of cases in each group to see how prevalence affects classification metrics.
  4. View classification metrics: See how your settings affect accuracy, sensitivity, specificity, and other metrics.

Interpreting the Results

The E2P Simulator provides several visualizations to help you understand the relationship between effect sizes and predictive utility:

In Binary Mode:

  • Overlapping distributions: Shows the separation between groups based on the effect size.
  • ROC curve: Shows the trade-off between sensitivity and specificity at different classification thresholds.
  • PR curve: Shows the trade-off between precision and recall at different classification thresholds.
  • Classification metrics: Provides quantitative measures of predictive performance.

In Continuous Mode:

  • Scatter plot: Shows the relationship between the predictor and outcome variables.
  • Distribution plot: Shows the distributions of the predictor variable for each group.
  • ROC curve: Shows the trade-off between sensitivity and specificity at different classification thresholds.
  • PR curve: Shows the trade-off between precision and recall at different classification thresholds.
  • Classification metrics: Provides quantitative measures of predictive performance.

Example Use Cases

Example 1: Planning a Clinical Trial

A researcher planning a clinical trial for a new treatment wants to determine what effect size they need to achieve to make the treatment clinically useful:

  1. Set the base rate to 0.5 (equal groups)
  2. Adjust the reliability to 0.8 (typical for clinical measures)
  3. Experiment with different Cohen's d values and observe that:
    • With d = 0.5, classification accuracy is only 64%
    • With d = 1.0, classification accuracy improves to 76%
    • With d = 2.0, classification accuracy reaches 92%
  4. Based on this exploration, determine that the intervention needs to achieve at least d = 1.5 to be clinically meaningful
  5. Use the Mahalanobis D calculator to determine how many smaller predictors would need to be combined to achieve this level of separation

Example 2: Interpreting Correlational Findings

A researcher has found a correlation of r = 0.3 between a psychological measure and an outcome of interest:

  1. Set the reliability to 0.7 (realistic for their measure)
  2. Set the base rate to 0.2 (the prevalence of the outcome in their population)
  3. Observe that:
    • The true correlation (corrected for reliability) is approximately r = 0.36
    • The classification accuracy is only 62%
    • The positive predictive value is 31%
  4. This helps understand that despite finding a "moderate" correlation, the practical predictive utility is limited
  5. Use the Multivariate RÂ² calculator to determine how many similar predictors would be needed to achieve more useful predictions
