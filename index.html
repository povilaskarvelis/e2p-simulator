Mahalanobis D Calculator

While the interactive graphs above can help determine the group separation needed for desired predictive performance, this separation can be achieved by combining multiple predictors. Mahalanobis D is a multivariate extension of Cohen's d that quantifies the overall separation between groups in multivariate space, accounting for the intercorrelations among predictors.

Mahalanobis D formula when all predictors have the same effect size and collinearity:

\[D = d \sqrt{\dfrac{p}{1 + (p-1)r_{ij}}}\]

Multivariate R² Calculator

While the interactive graphs above can help determine the correlation strength needed for desired predictive performance, this correlation can be achieved by combining multiple predictors. Multivariate R² quantifies the variance explained by a set of predictors, accounting for their intercorrelations.

Multivariate R² formula when all predictors have the same effect size and collinearity:

\[R^2 = \dfrac{pr^2}{1+(p-1)r_{ij}}\]
